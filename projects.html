Projects

This page highlights some of the tools, experiments, and side projects I’ve worked on. While most of my time (around 70%) is currently focused on Bladebound, I also enjoy experimenting with other ideas—ranging from local tooling to small games and automation projects.

Game Bot: Endfield

Discord Bot Automation Tool In Development

A game-specific Discord bot designed to keep servers updated automatically. It posts official news with excerpts, auto-posts new tweets while ignoring pinned posts, and provides an /endfield command with optional daily login automation via an endfield.js helper script.

Local LLM Chat (Mochi)

Local AI Chat Custom UI On Hold

A local AI chat interface with a custom UI, memory system, and extensible tooling. Built to feel responsive and comfortable for long sessions, and currently paused while focus is directed toward other projects.

Neon Dodger

Game Java Complete

A fast-paced Java arcade survival game. Dodge falling neon blocks as long as you can, featuring multiple difficulty modes, near-miss effects, screen shake, and a clean neon visual style.
